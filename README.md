## Functionalities of MyShelf
 - login  
 - registration 
 - adding a book 
 - viewing books 
 - adding or changing rate of books 
 - sorting a book with selected category
 
 ## Additionally admin has a few more functionalities: 
 - accepting or rejecting a book added by a user  
 - updating book data 
 - deleting a book
